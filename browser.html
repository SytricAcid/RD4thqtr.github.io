Popup boxes are those windows that pop out on the screen. They can appear before a page fully loads, as a response to an event, or before leaving a page. JavaScript has three kinds of popup boxes: alert box, user-input prompt box, and confirm box.

Sample Code

<script type="text/javascript"> function show_alert() { alert("Insert word here"); } </script> <input type="button" onclick="show_alert()" value="Show alert box">

The “If Statement” is used to check a condition and execute a statement only if the condition is true.

Sample Code

<script type="text/javascript"> function check() { if (document.example.answer.value == "Web Scripting") { document.write("You answered correctly!"); } } </script> <form name="example"> <input type="text" name="answer"> <input type="button" value="SUBMIT" onClick="check()"> </form>
